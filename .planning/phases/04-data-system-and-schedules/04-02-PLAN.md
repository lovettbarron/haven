Create `data/schedules/` directory.

Create `data/schemas/week.schema.json` using JSON Schema Draft 2020-12 with required fields from CONTEXT.md:
- `week` (integer), `iso_week` (string, e.g., "2026-W15"), `date_range` (start/end ISO dates)
- `theme` object: `name` (en/da), `tagline` (en/da), `hero_task` (string)
- `tasks` array: `id` (e.g., "W15-01"), `action`, `priority` (must_do/nice_to_do), `who` (child/father/together), `minutes`, `frequency` (daily/once/weekly), `bed`, `crop_id`, `prompt` (en/da)
- `growth_events` array: `crop_id`, `event`, `prompt` (en/da)
- `sowing_tasks` array: `crop_id`, `bed`, `slot`, `action`

Create `data/succession-calendar.json` FIRST (before weekly files) following Pattern 4 from research:
- Season 2026, zone 7b, Vejle
- Frost dates: last spring ~2026-05-10, first autumn ~2026-10-15
- Succession crops with multiple sowings:
  - Radish: sow every 2 weeks W16 through W32, harvest 3 weeks later (beds A, C, D, E rotating)
  - Lettuce baby mix: sow every 3 weeks W16 through W30, cut-and-come-again 3 weeks later
  - Spring onion: sow monthly W16, W20, W24, W28 (harvest 8 weeks later)
  - Peas: sow W16 + W22 (two plantings, harvest W24-W28 + W30-W34)
  - Bush beans: sow W20 + W24 (harvest W28-W32 + W32-W36)
- Non-succession continuous crops: strawberries (W23-W38), raspberries (W31-W40), kale (W26-W44), cherry tomatoes (W28-W40)
- gap_analysis section: for every week W22-W43, list which crops are harvestable. Flag any week with fewer than 2 harvestable crops. Target: no 2-week gap.
- Add 1-week buffer to all harvest estimates per research pitfall advice.

Create weekly schedule files W15-W29 (first half of season). For each week:
- ISO 8601 date ranges (2026 calendar: W15 = Apr 6-12, W16 = Apr 13-19, etc.)
- Theme name tied to the most exciting real event that week (per pitfall 5: name AFTER populating tasks). Examples:
  - W15: "Preparation Week" / "Forberedelses Uge"
  - W16: "Planting Day Week" / "Plantedags Uge"
  - W18: "Sunflower Sowing Week" / "Solsikke Uge"
  - W21: "Warm Friends Arrive" / "Varme Venner Ankommer" (tomato/cucumber transplant)
  - W23: "First Strawberry Week" / "Forste Jordbaer Uge"
- Tasks must include sowing_tasks from succession calendar for that week
- Every week must have at least one must_do task
- Hero task = the most exciting/visible task for the child
- Growth events with child-friendly bilingual discovery prompts ("Can you find..." / "Kan du finde..."), not botanical facts
- Reference crop_id values from Plan 01 crop files (lowercase-hyphen format)
- Tasks structured with who/minutes/frequency per CONTEXT.md locked decisions
